Demo 1: Proyecto Podcast Channel

Objetivo de la Demo: Presentar el trabajo hasta ahora realizado en el proyecto Podcast Channel, compartir el proceso y recibir feedback de compañeros y mentores.

El proyecto se encuentra en la última etapa del desarrollo, la fase de revisión y puesta online (80% completo).

Click Here to see the Project!

Desafío Presentado

"El desafío está enfocado en el maquetado de la landing page de un canal de Podcast, siguiendo las guías visuales de una interfaz de usuario otorgada y desarrollando funcionalidades de reproducción de contenidos, navegación, compatibilidad con múltiples exploradores y dispositivos".



Herramientas utilizadas

  • Visual Studio Code
  • HTML
  • CSS
  • SASS
  • Font Awesome
  • Google Fonts
  • Git
  • Trello


Planificación

Paso 1

Armado de Trello para organizar trabajo utilizando la Metodología Ágil de KANBAN

Paso 2

Escritura del código HTML para la página y simultáneamente escritura del CSS en Visual Studio Code

Pasos tomados:

  • 1) Se optó por comenzar con la vista Mobile First y armar un Media Query para la vista Desktop
  • 2) Se optó por maquetar cada sección de la página de manera completa (HTML + CSS para ambas vistas)
  • 3) Se comprobaba el correcto funcionamiento de cada sección
  • 4) Al finalizar cada sección, estas se subieron a GitHub documentando lo completado
  • 5) Se actualizaba el Trello con los pasos completados
Paso 3

Una vez escrito el código en CSS, se utilizó SASS para obtener de manera más sencilla el código necesario para adaptar las distintas funcionalidades escritas a los distintos Browsers.



Trabajo Futuro
